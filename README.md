# Multi-Strategy Quant Backtester

A modular Python backtesting engine built with **Backtrader** to compare popular technical analysis strategies using historical market data.

## 🚀 Overview
This project provides a clean framework for testing trading ideas against historical stock data. It currently supports three core strategies:
* **SMA Crossover**: Simple Moving Average "Golden Cross" logic.
* **RSI Mean Reversion**: Buying oversold conditions ($RSI < 30$) and selling overbought ($RSI > 70$).
* **Bollinger Bands**: Trading breakouts and mean reversion based on volatility bands.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** [Backtrader](https://www.backtrader.com/) (Backtesting engine)
* **Data Source:** [yfinance](https://github.com/ranaroussi/yfinance) (Yahoo Finance API)
